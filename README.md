# SRDT Internship Portal

## Overview

A full-stack internship portal built with Flask.

## Features

-   Student & Company Login
-   Profile Editing
-   Resume Upload
-   Profile Picture Upload
-   Internship Posting
-   Internship Applications
-   Saved Internships

## Tech Stack

-   Python
-   Flask
-   SQLAlchemy
-   MySQL
-   Bootstrap 5
-   Jinja2
-   Render
-   Git & GitHub

## Installation

``` bash
git clone https://github.com/raiadarsh0510/InternshipPortal.git
cd InternshipPortal
pip install -r requirements.txt
python run.py
```

## Environment Variables

    DATABASE_URL=your_database_url
    SECRET_KEY=your_secret_key

## Deployment

Start command:

    gunicorn wsgi:app
